---
title: "What is Rift?"
---

Rift is a multichain trading protocol with zero added fees.

Unlike other protocols that are secured by multisigs or staking, Rift uses a new primitive (multichain smart contracts) to automatically move funds between chains based on pre defined rules.

This unlocks 3 key benefits:

- **Non custodial** - Funds are controlled by smart contracts (protocol nodes running in cloud TEEs), and can be withdrawn at any time. Rift cannot steal or freeze user funds.
- **Largest size** - Trade 100m+ since Rift aggregates liquidity across different execution providers.
- **Lowest price** - Get the verifiably lowest rate across every execution venue. No added fees.

Rift's protocol code is 100% open source for anyone to verify, audit, or contribute to.
