---
title: "What is Rift?"
---

Rift is a multichain execution environment that lets you plug into liquidity on any chain.

Unlike other protocols that are secured by multisigs or staking, Rift uses hardware (cloud based TEEs) to automatically move funds between chains based on pre defined rules.

This unlocks previously impossible DeFi use cases, such as:

- Trade 100m+ for majors like BTC, ETH, USDC, USDT, HYPE. Non custodial, 0 fees.
- Earn yield from anywhere. Deposit 100m+ into bluechip vaults from anywhere.
- Advanced multichain orders. Limit orders, DCA, Stop loss/entry.
- Borrow any asset. Start with collateral on any chain.

Rift's [protocol code](https://github.com/riftresearch/tee-router) is 100% open source for anyone to verify, audit, or contribute to.
