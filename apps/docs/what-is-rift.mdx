---
title: "What is Rift?"
---

Rift is a protocol for multichain smart contracts. We let you plug into DeFi liquidity from any chain.

Unlike other protocols that are secured by multisigs or staking, Rift uses hardware (cloud based TEEs) to automatically move funds between chains based on pre defined rules.

This unlocks previously impossible DeFi use cases such as 100m+ market orders on major native assets, non custodial limit orders across chains, and depositing into vaults on any chain. Specifically, some key benefits of using Rift are:

- **Lowest price** - Get the verifiably lowest rate across every execution venue. No added fees.
- **Deepest liquidity** - Trade 100m+ since Rift aggregates liquidity across different execution providers.
- **Non custodial** - Funds are controlled by protocol nodes running in cloud TEEs, and can be withdrawn at any time. Rift cannot steal or selectively freeze user funds.

Rift's [protocol code](https://github.com/riftresearch/tee-router) is 100% open source for anyone to verify, audit, or contribute to.
