---
title: "Vaults"
---

A vault contract swaps a taker's starting asset into a vault, returning them a deposit receipt token. This enables takers to enter vaults on any chain from any starting asset. As an example, let's say a user wants to deposit Tron.USDT into a Morpho vault on Base, and earn in USDC.

![Vault flow](/images/use-cases-vault-flow.png)

Withdrawing works the same way, by redeeming the deposit receipt for deposited funds and swapping back to the desired starting asset if needed.
