# Railway Explorer Service

Railway service configuration for `explorer.rift.trade`.

Use `railway/explorer/Dockerfile` as the service Dockerfile path with the repository root as the build context.

The service builds and runs the Next.js app at `apps/explorer`.

Expected service settings:

- Builder: Dockerfile
- Dockerfile path: `railway/explorer/Dockerfile`
- Root directory: repository root
- Public domain: `explorer.rift.trade`
